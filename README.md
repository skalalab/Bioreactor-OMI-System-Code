# Peristaltic-Pump-GUI-Control
In-progress code and GUI for controlling the peristaltic pump for the SEED bioreactor


paristalttic_pump_control.ino is the code to be uploaded to an Arduino microcontroller

pump_control_timed.py has methods to control the pump through sending serial values to the Arduino,
and the methods are used by pump_interface.py

pump_interface.py is the in-progress GUI for pump control
